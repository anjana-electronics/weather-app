#  Weather App (Frontend)

A simple and responsive Weather Application built using frontend technologies. This app allows users to check real-time weather information for any city.



##  Features

*  Search weather by city name
*  Displays temperature, humidity, and weather conditions
*  Dynamic weather icons based on conditions
*  Responsive design (works on mobile & desktop)
*  Fast and user-friendly interface

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* OpenWeatherMap API (or any weather API you used)



##  Project Structure


weather-app/

---

##  How It Works

1. User enters a city name
2. App sends a request to the weather API
3. API returns weather data
4. Data is displayed on the UI

---

##  API Setup

1. Go to [OpenWeatherMap](https://openweathermap.org/api)
2. Create an account and get your API key
3. Replace the API key in your `script.js` file:

```javascript
const apiKey = "YOUR_API_KEY";
```



##  How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
